Tabela

ID Requisito Elicitação Modelagem Elo
R1 Apresentar micro-serviço (Facebook ou Google) e e-mail como forma de login e cadastro.
R2 O app deve permitir a escolha de interesses no primeiro acesso (no mínimo 5, obrigatoriamente), para que o Pinterest possa mapear os interesses do usuário.
R3 Ter um feed baseado nos interesses do usuário.
R4 Manter os interesses do usuário.
R5 O usuário deve ter a opção de salvar pins em um pasta
R6 O app deve possuir uma barra para pesquisa por texto.
R7 O App deve possibilitar a transferência de Pins entre pastas.
R8 O App deve permitir que o usuário impeça que determinado tipo de conteúdo seja mostrado em seu feed.
R9 O App deve obter vários dados de Pins bloqueados para identificar com precisão o tipo de conteúdo que o usuário não deseja visualizar.
R10 O app deve absorver a informação das categorias dos pins mais visualizados para usar nas próximas recomendações
R11 O app deve separar os pins por categorias
R12 O usuário deve ter a opção de avaliar pins e essa avaliação deve ser aberta a outros usuários
R13 O App deve possibilitar que o usuário reorganize os Pins salvos em diferentes subpastas.
R14 Ao clicar no pin o usuário deve conseguir ver mais informações sobre ele, como uma pequena ou grande descrição
R15 O usuário deve poder alterar seus interesses a qualquer momento.
R16 O usuário deve poder salvar seus pins no seu dispositivo
R17 Manter pastas.
R18 Manter subpastas.
R19 A aplicação deve conter um sistema de scroll infinito que possibilite ao usuário uma visualização mais orgânica do feed.
R20 A plataforma deve disponibilizar uma forma de pesquisa por imagem.
R21 O usuário deve poder ser redirecionado para o link da imagem de um pin, caso haja um.
R22 O App deve sugerir Pins sobre assuntos pesquisados recentemente pelo usuário.
R23 O usuário deve conseguir compartilhar pins com outros usuários
R24 O usuário deve conseguir compartilhar pins en redes sociais
R25 A aplicação deve colocar nos pins a opção de "comentários"
R26 Ao pesquisar sobre algo, o Pinterest deve mostrar opções de filto sobre o assunto, para que o usuário apenas clique em um e receba um resultado mais selecionado
R27 O Pinterest deve oferecer a opção de seguir usuários
R28 Os pins devem oferecer informações sobre o motivo dele estar aparecendo para o usuário.
R29 O usuário deve poder ocultar denunciar pins.
R30 O aplicativo deve apresentar um navegador interno para redirecionamento de links presentes em pins.
R31 O app deve possuir a função de compartilhar a edição de pastas com outros usuários.
R32 O app deve oferecer mais conteúdo relacionado às atividades recentes.
R33 As receitas devem ser pins especiais que ofereçam tempo, porções e ingredientes para instigar o usuário a fazê-la.
R34 O usuário deve ter o ferramental necessário para realizar as etapas de uma receita sem precisar sair do app
R35 A interface do usuário deve ser agradável para melhorar as experiências e torná-las excelentes.
R36 A rede social deve incluir sempre novos conteúdos sobre os temas mais pesquisados em busca de mantê-los atualizados, mas também procurar abranger temas diferentes para atrair novos públicos específicos, como os praticantes de esportes, por exemplo.
R37 O Aplicativo deve ser capaz de extrapolar dados de pesquisa de outras fontes para trazer itens mais relevantes para o usuário.
R38 O Aplicativo deve identificar a perda de interesse do usuário por um tema pela frequência de pesquisas.
R39 O app deve ser de rápida aprendizagem.
R40 O usuário deve poder enviar mensagem para outros usuários.
R41 O usuário deve receber notificações sobre novidades no app.
R42 O usuário deve poder editar seu perfil.
R43 Os pins que se referem a artigos devem mostrar seu autor e descrição.
R44 Os pins que se referem a aplicativos devem mostrar a opção de instalar.
