    Versionamento

    Versão Modificação
    1.0 A Primeira versão dos léxicos foram feitas por todo o grupo, para as primeiras entregas referentes ao módulo, antes das entrega final. A Cada versão, obteve-se adição de termos para composição dos léxicos, de forma que essa versão reflita bem como foram montados
    2.0 A Segunda versão busca atender a propostas de composição dos léxicos conforme orientação, com adição das classificações e reformulação de alguns léxicos quanto a noção e impacto, feita por todo o grupo

    Introdução

    É comum em meios sociais que existam termos próprios relacionadas às ações e contextos do cotidiano. Dessa forma, surgem os léxicos: termos que descrevem os símbolos de uma linguagem e que denotam ações, atores ou estados que possuem representação única quando relacionada a um contexto específico.

    Léxicos

    L1

    Pin

    Versão 1.0

    Noção
    Uma imagem ou conteúdo exibido dentro do Pinterest.
    É a principal moeda de interação do usuário.
    A interação ocorre a partir deste construto.
    Os pins estão presentes dentro do feed.
    Os pins possuem interesses relacionados a eles.
    Os pins são salvos em pastas.
    Impacto
    O Pin pode ser compartilhado, comentado, pinado em uma pasta ou subpasta e baixado.
    O usuário consome pins do Pinterest para buscar inspiração.
    O usuário pode denunciar um pin caso ele seja contra as Diretrizes da Comunidade do Pinterest.
    O usuário pode marcar o Pin como já experimentado.
    Dentro de um Pin é possível pesquisar por itens visualmente semelhantes.
    Dentro de um Pin é possível copiar o link da imagem que redireciona para o Pinterest.
    Sinônimos
    Pins, Imagem, Imagens, Foto, Fotos, Post, Posts, Postagem

    L2

    Pinar

    Versão 1.0

    Noção
    É o ato do usuário salvar um pin em um pasta para poder acessar mais tarde ou para categorizar seus pins.
    Impacto
    O usuário pode pinar um pin de que ele goste.
    O usuário guarda seus pins em um pasta.
    O usuário pode acessar posteriormente os pins salvos em suas pastas.
    A interação de pinar aumenta a classificação de relevância do interesse relacionado ao pin pinado, o que faz com que mais pins relacionados a este interesse apareça no feed do usuário.
    Sinônimos
    Pinado, Salvar, Salvo, Salva

    Pasta

    Versão 1.0

    Noção
    É uma coleção de pins, contendo título para poder se diferenciar de outras coleções.
    É utilizado pelos usuários para organizar os seus pins pinados.
    Pode possuír sub-conjuntos denominados subpastas.
    Impacto
    O usuário pina pins em um pasta.
    O usuário pode criar subpastas para organizar os pins da pasta.
    O usuário pode convidar outros usuários para colaborar com outros pins dentro do pasta.
    Uma pasta pode ser seguida por outro usuário para ser acompanhada.
    Uma pasta pode ser recomendada para usuários por determinados interesses.

    Subpasta

    Versão 1.0

    Noção
    É uma coleção de pins interna a uma pasta
    Uma pasta de Pins dentro de uma outra pasta.
    Impacto
    O usuário pina pins em uma subpasta.
    O usuário pode criar várias subpastas dentro de uma mesma pasta.
    O usuário que colabora com a pasta também colabora com suas subpastas.
    Os Pins do usuário ficam organizados por subtemas pertencentes ao tema da pasta.
    O conteúdo dos pins em uma subpasta costuma possuir alguma relação com o conteúdo da pasta, mas formam um subconjunto próprio dentro da temática maior da pasta
    É utilizado pelos usuários para organizar os seus pins dentro de uma pasta.
