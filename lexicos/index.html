Pin

Noção
Uma imagem exibida dentro do Pinterest. É a funcionalidade principal, tendo tudo ao seu redor. A interação ocorre a partir deste construto. Podendo ser COMPARTILHADO, COMENTADO, pinado e outros.
Impacto
O usuário consome pins do Pinterest para buscar inspiração.

Pinar

Noção
É o ato do usuário guardar um pin em um board para poder acessar mais tarde ou para categorizar seus pins.
Impacto
O USUÁRIO pode pinar um pin de que ele goste.

Board

Noção
É uma coleção de pins, tendo título para poder se diferenciar de outras coleções. É utilizado pelos usuários para organizar os seus pins pinados. Pode possuír sub-conjuntos.
Impacto
O usuário pina pins em um board.

Pinterest

Noção
Serviço de compartilhamento de imagens (pins) em que os usuários acessam em busca de inspiração e também de ver imagens relacionadas aos seus interesses.
Impacto
O PINTEREST é muito legal! =)

Interesse

Noção
Algo em que o usuário do PINTEREST está interessado.
